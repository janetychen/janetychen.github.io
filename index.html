Athens Game Jam

Winning Game Design of the Athens Game Jam

                           

Michael Melatti and Janet Chen entered Athens Game Jam, a video game competition held in Athens Georgia. Our team of two took first place at this competition with our entry called Double Take. Double Take takes advantage of Google CardBoard VR as well as physical board game objects. In the game, the user has to memorize objects position in 3D space. The app will then prompt the user to start placing objects on the board and the user must rely on memory to get as close as possible. The phone will then buzz signaling that time is up and the phone will then display the correct answer so the users can calculate their score. This entire game was created in 2 days.

  • March 2016 - March 2016
  • Client: Personal Project
  • Category: Software Design In Unity

Touch'N-Go

New Technologies Created For the Unity Engine

           

This project consisted of creating a new drawline technology within the Unity Game Engine. The user can use a touch screen interface to draw lines with physics properties in real time within the game. This is accomplished by creating a linked list of touch inputs and instantiating box-colliders and using euclidian geometry to place the box-colliders and optimize performance.

  • Dec 2014 - Jan 2015
  • Client: Personal Project
  • Category: Software Design In Unity

Research

Research published while in undergrad

Research involves creating new methods for mediated learning and instruction within virtual reality. Using Unity 3D a VR classroom application was created with two interfaces. One interface allows the instructor creation of unique lectures within the VR environement. The other interface is a view interface for students to view the lecture in VR or on a desktop interface. VR was used to eliminate the spatial barriers associated with engineering concepts. In building this tool, Michael developed a suite for recording objects within a scene as well as captureing the teachers audio and creating a playback interface for students. Michael also created VR tools for the instructor to use in recording.

These projects will takes advantage of Unity3D, SteamVR, C#, HTC Vive, Google Cardboard, and mobile development.

  • Aug 2016 - May 2017
  • Client: UGA/Dr. Kyle Johnsen, Dr. Savadatti, Dr. Pidaparti
  • Category: Undergrad Research
